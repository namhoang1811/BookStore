# 📚 BookStore Management System

Hệ thống quản lý nhà sách trên nền tảng Console, được tối ưu hóa hiệu năng với cấu trúc dữ struct và phát triển trên IDE Visual Studio 2026.

## 📖 Giới thiệu

Đây là một bài tập lớn môn "Cấu trúc dữ liệu và giải thuật" của nhóm sinh viên đại học Mở Hà Nội - HOU. BookStore là một dự án quản lý bán lẻ sách (Console Application) được thiết kế để giải quyết bài toán vận hành thực tế của các nhà sách vừa và nhỏ.

## 🚀 Tính năng nổi bật

### Phân hệ Quản trị (Admin)

*   Quản lý kho hàng (Inventory): Thêm, xóa, sửa thông tin sách.
    
*   Nhập hàng (Restock): Cập nhật số lượng tồn kho nhanh chóng.
    
*   Báo cáo: Xem danh sách sách bán chạy, doanh thu theo ngày.
    
*   Quản lý dữ liệu: Tự động lưu trữ dữ liệu vào File hệ thống khi thoát.
    
### Phân hệ Khách hàng (User)

*   Tìm kiếm thông minh: Tìm theo Tên, Tác giả (Partial Search) hoặc ISBN (Exact Match).
    
*   Giỏ hàng (Shopping Cart): Thêm/bớt sản phẩm, xem tạm tính.
    
*   Thanh toán: Xử lý đơn hàng, kiểm tra tồn kho thời gian thực, xuất hóa đơn.

## 🧪 Kiểm thử (Testing)

Dự án tích hợp sẵn bộ Unit Test để đảm bảo tính đúng đắn của các thuật toán tính tiền và quản lý kho.

Để chạy test:

1.  Trong Visual Studio, mở menu BookStore \> Test BookStore.Tests
    
2.  Nhấn Run All Tests (hoặc biểu tượng Play màu xanh).
    
3.  Kết quả kiểm thử sẽ hiển thị trạng thái Pass/Fail cho từng hàm.
